
List of related/similar/competing tools to Caret. 

Coding IDEs:
- Cursor/VSCode/Any modern IDE

Chat/AI:
- ChatGPT Canvas

RAG / LLMWiki / Second Brain:
- NotebookLM
	- Chat centric, lack of focus on writing
	- https://notebooklm.google.com
- Atlas
	- Chat with sources
	- Notes over sources
	- Graph
	- No writing experience
	- https://www.atlasworkspace.ai
- Cabinet
	- Not focused on writing
	- https://github.com/hilash/cabinet
- Wikiwise
	- https://github.com/TristanH/wikiwise/
- Hatch
    - https://hatch.so
- nashsu's LLM wiki
	- https://github.com/nashsu/llm_wiki

Document editors:
- Google Docs
- Amber
	- For professional writers
	- Close to Google Docs with Agents
	- https://www.withamber.com

Obsidian:
- Obsidian copilot plugin
	- Most popular AI obsidian plugin
	- https://www.obsidiancopilot.com
	- https://github.com/logancyang/obsidian-copilot
- Obsidian Smart Composer plugin
	- Unmaintained
	- https://github.com/glowingjade/obsidian-smart-composer

Scientific / Academic research:
- Google Co Scientist 
	- Focused on academic research
	- https://research.google/blog/accelerating-scientific-breakthroughs-with-an-ai-co-scientist/
- Azure Discovery
	- Focused on academic research
	- https://azure.microsoft.com/en-us/blog/transforming-rd-with-agentic-ai-introducing-microsoft-discovery/
